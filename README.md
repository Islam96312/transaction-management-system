# Transaction Management System

## Installation Instructions
1. Clone the repository: `git clone repo-url`
2. Navigate to the directory: `cd transaction-management-system`
3. Create a `.env` file using the provided `.env.example`
4. Install dependencies: `pip install -r requirements.txt`

## Project Structure
- `backend/`: Contains all backend-related code
- `frontend/`: Contains all frontend-related code

## API Documentation
- Refer to the FastAPI documentation for details on available endpoints.

## Running the Project
### For Backend:
1. Navigate to the backend directory.
2. Run the application: `uvicorn main:app --host 0.0.0.0 --port 8000`

### For Frontend:
1. Navigate to the frontend directory.
2. Run the application: `npm run dev`
